## Use as Extension

Deze repository kan worden toegevoegd als **extensie** in MakeCode.

* open [https://makecode.microbit.org/](https://makecode.microbit.org/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/ralphcrutzen/nachtlampje** and import

## Dit project bewerken ![Statusbadge bouwen](https://github.com/ralphcrutzen/nachtlampje/workflows/MakeCode/badge.svg)

Om deze repository te bewerken in MakeCode.

* open [https://makecode.microbit.org/](https://makecode.microbit.org/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/ralphcrutzen/nachtlampje** and click import
